Git

  • git -v - Проверить установку программы Git.
  • git branch - Создать верку в репозиторие.
  • git branch -a - Вывод всех веток. Локальных и удаленных
  • git checkout - Переключение между ветками.
  • git status - Проверка включения изменений в будущий коммит.
