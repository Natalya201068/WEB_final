Git

  • git pull-команда,отправляющая изменения с удаленного репозитория на локальный.
  • git clone "ссылка с github"-команда позволяющая скопировать удаленный директорий на компьютер.
  • git push-команда,которая отправляет изменения с локального репозитория на удаленный.
  • git commit -m "текст коммита"-команда для записи изменений в репозиторий Git.
  • git status-команда,позволяющая отследить изменения в конкретной папке.
