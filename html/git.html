Git

  • git clone - Клонирует удаленный репозиторий на локальный компьютер.
  • git add - Добавляет указанный файл в индекс для последующего коммита.
  • git log- Oтображает историю коммитов.
  • git status- Показывает статус рабочего каталога и индекса
  • git branch- Создает, просматривает или удаляет ветки
